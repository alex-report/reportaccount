Social Media Report Tool - FAQ

What is Social Media Report Tool?

The Social Media Report Tool is an efficient and reliable solution designed to help users report pages across various social media platforms.

How to use?

To create a new order:

  • Open page "Neworder"
  • Select platform type
  • Select service
  • Enter the URL of the page you need to report
  • Enter the quantity you want
  • Submit
  • Pay for your order

How long does the reporting process take? When can the page be banned?

  • For batches of less than 10,000, it is typically completed within a few hours.
  • Whether a page gets banned depends on the volume of orders submitted. Generally, a higher number of reports increases the probability of a successful ban.
  • If the page is not banned after a single order is completed, you can submit a new order to increase the success rate.
  • After order completion, please allow 1-2 days to observe if the page is banned, as there might be a delay in the reporting effect.

You can check the reporting progress of your orders in real-time on the Orders page.
